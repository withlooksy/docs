## 🏗️ Technical Setup

### Local Development

```bash
# Install Mintlify CLI
npm i -g mintlify

# Navigate to docs directory
cd ~/Desktop/looksy-docs

# Start local server
mintlify dev

# Open browser to http://localhost:3000
```

### Deployment

Mintlify auto-deploys when connected to your GitHub repository:

1. Push changes to repository
2. Mintlify automatically builds and deploys
3. Changes live within minutes

## 📝 Content Guidelines

When adding or updating content:

1. **Write for merchants** - Practical, results-focused
2. **Use conversational tone** - Friendly but professional
3. **Include specifics** - Numbers, data, examples
4. **Be transparent** - Honest about limitations
5. **Provide next steps** - Clear calls to action
6. **Link related content** - Internal cross-references
7. **Use visual elements** - Cards, accordions, callouts
