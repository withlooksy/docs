## Looksy Works with All Shopify Themes

Looksy is built to integrate seamlessly with any Shopify theme, whether you're using a popular theme from the Theme Store or a custom-built theme.

## Integration Methods

### Method 1: App Embed (Recommended)

The **app embed** is the easiest and most reliable way to add Looksy to your store.

**Benefits:**
- Works with any theme automatically
- No code changes required
- Updates automatically
- Theme updates won't break it
- Takes 30 seconds to enable

**How to enable:**

1. Go to **Online Store → Themes** in your Shopify admin
2. Click **"Customize"** on your active theme
3. In the theme editor, find **"App embeds"** in the left sidebar
4. Locate **"Looksy Virtual Try-On"** and toggle it **ON**
5. Click **"Save"**

<Check>
The try-on button will now appear on all enabled product pages automatically.
</Check>

### Method 2: App Block (Advanced Control)

For more control over button placement, use the **app block** method.

**Benefits:**
- Precise placement control
- Custom positioning on product pages
- Can adjust styling and layout
- Per-template customization

**How to add:**

1. Go to **Online Store → Themes → Customize**
2. Navigate to a **Product page** template
3. Click **"Add block"** or **"Add section"**
4. Select **"Looksy Virtual Try-On"** from Apps
5. Drag the block to your preferred location
6. Click **"Save"**

<Tip>
Best placement: Near the "Add to Cart" button, above product description, or in the product details section.
</Tip>

## Customizing Button Appearance

### Theme Editor Customization

In the Shopify theme editor:

1. Click on the **Looksy block** on your product page
2. Adjust available settings:
   - Button text (e.g., "Try it on", "See it on you")
   - Button color (uses theme colors by default)
   - Button size
   - Alignment (left, center, right)

### CSS Customization (Advanced)

For custom styling beyond theme editor options:

```css
/* Target Looksy button */
.looksy-try-on-button {
  background-color: #0D9373 !important;
  border-radius: 8px !important;
  padding: 12px 24px !important;
  font-weight: 600 !important;
}

/* Hover state */
.looksy-try-on-button:hover {
  background-color: #0A7A5E !important;
}
```

Add this to your theme's custom CSS section (Theme → Edit code → Assets → theme.css).

<Warning>
Custom CSS may be overridden by theme updates. Document your changes for easy reapplication.
</Warning>

## Multi-Template Setup

If your store uses multiple product page templates:

### For Each Template:

1. Navigate to the specific template in the theme editor
2. Add the Looksy app block to that template
3. Position it consistently across templates
4. Save each template

**Common templates:**
- Default product template
- Featured product template
- Product quick view
- Collection product cards (if applicable)

### Pre-Launch Checklist

- [ ] Clicking button opens try-on modal
- [ ] Upload flow works smoothly

Test on:

## Troubleshooting Integration Issues

<AccordionGroup>
  <Accordion title="Button doesn't appear on product pages">
    **Possible causes:**
    - App embed not enabled
    - Product not enabled in Looksy dashboard
    - Theme conflict or JavaScript error

    **Solutions:**
    1. Verify app embed is ON in theme settings
    2. Check product is enabled in Looksy dashboard
    3. Clear browser cache and test in incognito mode
    4. Check browser console for JavaScript errors
  </Accordion>

  <Accordion title="Button placement looks wrong">
    **Possible causes:**
    - Theme CSS conflicts
    - Default positioning doesn't fit your layout

    **Solutions:**
    1. Switch from app embed to app block for manual placement
    2. Adjust positioning in theme editor
    3. Add custom CSS to fine-tune appearance
  </Accordion>

  <Accordion title="Button works on desktop but not mobile">
    **Possible causes:**
    - Mobile template doesn't include app block
    - CSS hiding button on mobile
    - JavaScript not loading on mobile

    **Solutions:**
    1. Use app embed (handles mobile automatically)
    2. Check mobile-specific theme settings
    3. Test on actual mobile device, not just browser resize
  </Accordion>

  <Accordion title="Try-on modal doesn't open">
    **Possible causes:**
    - JavaScript conflict with other apps
    - Browser blocking modals
    - Theme JavaScript errors

    **Solutions:**
    1. Disable other apps temporarily to isolate conflict
    2. Check browser console for errors
    3. Test in incognito mode
    4. Contact support with error details
  </Accordion>
</AccordionGroup>
