## Looksy Works with All Shopify Themes

Looksy is built to integrate seamlessly with any Shopify theme, whether you're using a popular theme from the Theme Store or a custom-built theme.

## How to Enable Looksy

### App Embed (Only Method)

Enable Looksy in 30 seconds with the app embed:

1. Go to **Online Store → Themes** in your Shopify admin
2. Click **"Customize"** on your active theme
3. In the theme editor, find **"App embeds"** in the left sidebar
4. Locate **"Looksy Try-On Button"** and toggle it **ON**
5. Click **"Save"**

<Check>
The try-on button will now appear on all product pages automatically. Works with all themes.
</Check>

## Customizing the Button

You can customize the button appearance in your Looksy dashboard:

**What you can customize:**
- Button text (default: "Try On")
- Button colors
- Button corner position (top left or top right)
- Button styling (fonts, size)

**How to customize:**

1. Go to your **Looksy dashboard**
2. Click **"Customize Button"**
3. Adjust text, colors, and corner position
4. Click **"Save"**

<Note>
The button appears on product images automatically. You can only change the styling and corner position—not the location.
</Note>

## Theme Compatibility

### Supported Theme Types

Looksy works with:
- **Shopify Theme Store themes** (Dawn, Debut, Brooklyn, etc.)
- **Custom themes** (built by developers)
- **Sectioned themes** (Shopify 2.0 and newer)
- **Vintage themes** (pre-2.0 themes)
- **Headless/custom frontends** (with API integration)

### Theme Requirements

**Minimal requirements:**
- Shopify-compatible theme
- Product pages enabled
- JavaScript enabled (standard for all Shopify themes)

**No special requirements for:**
- Theme version
- Theme framework
- Custom code
- Developer involvement

## Works with All Themes

Looksy works automatically with:
- Dawn, Debut, Brooklyn, and all Shopify Theme Store themes
- Custom themes built by developers
- Shopify 2.0 and older themes
- No special setup or adjustments needed

<Check>
Enable the app embed once and it works on all themes. No theme-specific configuration required.
</Check>

## Testing Your Integration

After enabling the app embed:

1. Visit a product page on your live store
2. Click the "Try On" button
3. Upload a selfie and verify results

<Check>
If the button appears and try-on works, you're done. It works automatically on desktop, mobile, and tablet.
</Check>

### Theme Updates

When updating your theme:
- App embed automatically persists
- No action needed after theme updates

## Troubleshooting

<AccordionGroup>
  <Accordion title="Button doesn't appear on product pages">
    **Solution:**
    1. Verify app embed is ON in theme settings (Online Store → Themes → Customize → App embeds)
    2. Check product is enabled in Looksy dashboard
    3. Clear browser cache
  </Accordion>

  <Accordion title="Button styling looks wrong">
    **Solution:**
    Go to Looksy dashboard → Customize Button and adjust colors, text, and corner position to match your brand.
  </Accordion>

  <Accordion title="Try-on doesn't work">
    **Solution:**
    1. Test in incognito mode
    2. Try a different browser
    3. Contact support@looksy.ai with details
  </Accordion>
</AccordionGroup>
