## How do I customize the Looksy button?

Open the Looksy app embed in Shopify's theme editor. The **Customize button** action in the Looksy dashboard can take you to that surface when the active theme and plan are ready.

## Open the controls

1. In Shopify admin, go to **Online Store → Themes**.
2. Select **Customize** for the intended theme.
3. Open **App embeds → Looksy Try-On Button**.
4. Enable the embed if necessary.
5. Change the supported settings and save the theme.

## Available settings

The current extension includes controls for button position, text, text and background color, icon, typography, border radius, padding, offsets, and result-thumbnail radius. The exact controls shown can vary with the installed extension version.

Start with the smallest useful changes:

- use short, literal button text;
- maintain sufficient color contrast;
- select a corner that does not cover important imagery or gallery controls;
- adjust spacing only as needed for the current theme;
- retain an icon only when it remains understandable and uncluttered.

## Verify before publishing

Test representative product templates, variants, image aspect ratios, desktop widths, and mobile devices. Confirm the button remains visible, readable, keyboard reachable, and separate from other overlays.

<Note>
Theme behavior can change after a theme update or template edit. Re-test Looksy whenever the storefront theme changes materially.
</Note>

For placement or compatibility problems, use the [theme integration guide](https://withlooksy.com/docs/integration/theme-setup) and capture the theme name, template, product URL, screenshot, and device/browser for support.
