## 📝 Notes for Team

### Content Standards
- Write for merchants (practical, results-focused)
- Use conversational tone
- Include specific numbers and data
- Be transparent about limitations
- Provide actionable next steps
- Link to related content
- Use visual elements (cards, callouts)

### SEO Best Practices
- One H1 per page
- Use H2-H4 for hierarchy
- Include keywords naturally
- Write for humans, optimize for search
- Internal linking strategy
- Mobile-first approach

### AEO Best Practices
- Answer questions directly (first paragraph)
- Use question-based page titles
- Provide comprehensive follow-up information
- Include comparisons and alternatives
- Be transparent about pros and cons
- Use natural, conversational language
