## Will Looksy affect storefront performance?

Any storefront feature can affect loading, responsiveness, or network activity. The practical answer depends on your theme, other apps, device mix, connection quality, and where the feature is loaded.

Measure the actual effect on your store instead of relying on a blanket performance claim.

## How should I test performance?

1. Record a baseline on representative product pages before enabling Looksy.
2. Test the same pages after enabling it.
3. Use both lab tools and real-user data where available.
4. Check mobile devices and slower connections, not only a fast desktop.
5. Test the page before opening try-on and while completing the try-on flow.

## What should I measure?

- Largest Contentful Paint (LCP)
- Interaction to Next Paint (INP)
- Cumulative Layout Shift (CLS)
- JavaScript execution and long tasks
- transferred bytes and request count
- visible delays, layout movement, or blocked interaction

## What if I see a regression?

Confirm that the difference is repeatable, then check automatic product-image placement, other app scripts, duplicate installation, network failures, and device-specific behavior. Record the affected URL, theme, browser, device, and measurements when contacting support.

<Note>
  A single Lighthouse run is not definitive. Compare multiple runs and use field data when enough traffic is available.
</Note>
