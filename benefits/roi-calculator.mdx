## What information do I need to estimate ROI?

Gather these inputs for the same period:

| Input | Where to get it |
| --- | --- |
| Sessions and orders | Shopify analytics |
| Conversion rate | Orders divided by relevant sessions |
| Average order value | Shopify analytics |
| Contribution margin | Your finance records |
| Return rate and cost per return | Your returns and fulfilment records |
| Looksy cost | Your Looksy billing information |

## Estimate incremental contribution

```text
baseline orders = sessions × baseline conversion rate
test orders = sessions × test conversion rate
incremental orders = test orders - baseline orders
incremental contribution = incremental orders × contribution per order
```

## Estimate return-related savings

```text
avoided returns = expected returns - observed returns
return savings = avoided returns × cost per return
```

Only attribute a change to Looksy when the test design supports that conclusion.

## Calculate ROI

```text
measured benefit = incremental contribution + documented return savings
net benefit = measured benefit - Looksy cost
ROI (%) = net benefit / Looksy cost × 100
```

<Warning>
  This worksheet is an estimation method, not a promise of results. Use your own data and include uncertainty in the result.
</Warning>

## Run a sensitivity check

Calculate at least three scenarios:

- **Conservative:** lower measured benefit and full costs.
- **Expected:** the best-supported values from your test.
- **Upper range:** a plausible upside case, clearly labelled as an estimate.

If the decision changes substantially between scenarios, collect more data before expanding the rollout.
