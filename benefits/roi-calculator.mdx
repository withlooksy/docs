## Breakeven Analysis

### How Many Try-Ons to Break Even?

**For $99/month plan:**

```
Required monthly benefit = $99
Average benefit per try-on = $3-5 (typical)
Required try-ons = $99 / $4 = 25 try-ons
```

**With 35% engagement rate:**
```
Required visitors = 25 / 0.35 = 72 visitors/month
```

**Conclusion:** You only need 72 monthly visitors to break even. Most stores have thousands.

<Check>
Virtually any store with regular traffic will see positive ROI from virtual try-on.
</Check>
