## Can virtual try-on reduce returns?

Virtual try-on may help shoppers form clearer expectations before ordering. Its effect depends on the product, the return reason, the quality of the experience, and store operations. Looksy does not guarantee a return-rate reduction.

## Which return reasons should I examine?

Separate return reasons before judging the result. Virtual try-on is unlikely to affect every cause of a return, such as fulfilment errors, defects, late delivery, or a shopper ordering multiple sizes intentionally.

Useful categories include:

- appearance differed from expectation;
- colour or styling uncertainty;
- fit or sizing concern;
- product quality or damage;
- fulfilment or delivery issue;
- changed mind or other reason.

## How should I measure the impact?

1. Record return reasons consistently.
2. Compare similar products and order cohorts.
3. Wait for the applicable return window to close.
4. Report order and return counts with percentage changes.
5. Note promotions, product changes, seasonality, and policy changes.

<Warning>
  Do not attribute every change in returns to Looksy. Pair return data with try-on usage and a suitable comparison group.
</Warning>

## What else can reduce avoidable returns?

Use accurate product photography, clear sizing information, complete descriptions, visible policies, and responsive customer support alongside virtual try-on.
