## How can an ecommerce store reduce returns?

Start with return reasons, not a generic benchmark. Different causes require different fixes, and a single feature will not address every return.

## Match the intervention to the reason

| Return reason | Areas to investigate |
| --- | --- |
| Appearance differed from expectation | Photography, colour, description, virtual try-on |
| Size or fit concern | Size guide, measurements, fit notes, support |
| Quality concern | Product detail, materials, quality control |
| Wrong or damaged item | Picking, packing, carrier, quality control |
| Delivery issue | Estimates, tracking, carrier performance |

## Where can Looksy help?

Looksy may help some shoppers evaluate appearance before buying. Test it on products where visual uncertainty is a meaningful return reason. It does not replace accurate sizing, product information, or fulfilment controls.

## How should I measure improvement?

1. Standardize return-reason collection.
2. Choose comparable product and order cohorts.
3. Wait until the return window closes.
4. Compare counts and rates, not percentages alone.
5. Review margin and customer-experience effects alongside return rate.

<Note>
  Looksy does not guarantee a specific reduction in returns. Results vary and should be established with your own data.
</Note>
