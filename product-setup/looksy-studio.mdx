---
title: "Create Model Images With Looksy Studio"
description: "Generate an on-model product image from existing product imagery and a Looksy or uploaded model photo, then add the result to Shopify."
---

Looksy Studio creates an on-model product image using the product's existing imagery and either a bundled Looksy model or your own model photo. You can review the generated result before adding it to the Shopify product's images.

<Note>
The **Model images** card appears in a product's Looksy configuration when that product has a featured image and the feature is available in your Looksy environment.
</Note>

## Generate a model image

1. In **Shopify Admin**, open **Looksy**.
2. Select **Products** and open the product you want to configure.
3. Find **Model images**.
4. Choose one of the bundled Looksy models, or select **Upload** and provide your own model photo.
5. Select **Generate model image**.
6. Review the generated image.

Each successfully generated model image uses **1 credit**.

<Warning>
A model image is generated content. Review the garment, model, proportions, colours, details, and brand suitability before using it in your catalog.
</Warning>

## Add the result to Shopify

After reviewing a successful result:

1. Select **Add to product images**.
2. Wait for Looksy to confirm **Added to product images**.
3. Review the product in Shopify and confirm the new image is present.
4. Adjust its ordering or remove it through Shopify's product media tools when needed.

Generating and adding are separate actions. Closing the review without selecting **Add to product images** does not intentionally add the generated result to the product's Shopify media.

## If generation fails

- Try another model photo.
- Confirm that the product still has usable product imagery.
- Use a clear model photo without visual obstructions.
- Retry from the **Generated model image** dialog.

If the image is generated but cannot be added to Shopify, keep the result open and retry **Add to product images**. Contact support if the save continues to fail.
