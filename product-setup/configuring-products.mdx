## What can I configure for each product?

Each enabled product can have its own try-on imagery, appearance-specific setup, size chart, direct try-on link, and model-image workflow. Open **Looksy → Products**, then select the product you want to configure.

## Choose try-on images

In **Try-on Images**, choose up to six useful product images with different angles or details. These images are inputs to the virtual try-on; they are not extra storefront gallery positions.

Prefer images that show the product clearly and represent the item the shopper will receive. See [Image Requirements](https://withlooksy.com/docs/product-setup/image-requirements) before combining weak or inconsistent source images.

## Configure appearances and sizes

Looksy treats visual options and sizes differently:

- Visual options such as colour, pattern, or material can keep separate try-on imagery.
- Sizes within the same appearance share one try-on-image setup.
- If older configuration exists separately for several sizes, Looksy asks which size's setup to keep before unifying them.

Review the selected appearance before saving. Do not assume changing one appearance updates every colour or material.

## Review size recommendations

The **Size recommendations** card lets you review or edit the product's size chart and, where supported, map recommendations to available Shopify variants. The generated try-on image itself does not infer a shopper's size.

See [Find Your Size](https://withlooksy.com/docs/shopper-features/size-recommendations) for the shopper flow and accuracy boundaries.

## Use product defaults for future items

Under **Settings → Products**, you can:

- turn try-on on automatically for products added in future;
- use each new product's featured image by default; or
- turn that option off so Looksy can combine up to four gallery images.

These defaults do not change existing products. Review existing items individually or use the current collection bulk controls.

<Note>
  Always save and test a representative product page after changing imagery, appearance, size-chart, or targeting settings.
</Note>
