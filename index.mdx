Looksy is an AI virtual try-on app for Shopify. A shopper uploads a photo and Looksy generates a visual preview with an enabled product. The preview can help with appearance and style decisions, but it is not a sizing or fit guarantee.

On eligible product pages, Looksy automatically places the **Try On** button over the main product image and shows the generated preview in that same image area. The Shopify app embed—the theme-wide on/off switch—enables this behavior; it is not a block you place in the page layout.

## What Looksy does

Use these docs to install Looksy, choose eligible products, turn on the storefront experience, test the shopper flow, review usage, and troubleshoot known issues. Begin with a small product set and verify the experience on your own storefront before making it broadly available.

## Use the docs by task

- **Product setup:** Choose an initial product set and review the documented image requirements.
- **Storefront setup:** Turn on the app embed, adjust the product-image button if needed, and test desktop and mobile storefronts.
- **Analytics:** Review the metrics available in Looksy and use the experiment guides without assuming a guaranteed outcome.
- **Troubleshooting:** Check button visibility, product eligibility, image inputs, and device/browser details in a consistent order.
