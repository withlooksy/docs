Looksy is an AI virtual try-on app for Shopify. A shopper uploads a photo and Looksy generates a visual preview with an enabled product. The preview can help with appearance and style decisions, but it is not a sizing or fit guarantee. **Find Your Size** is a separate feature that compares measurements entered by the shopper with the merchant's product size chart.

On eligible product pages, Looksy automatically places the **Try On** button over the main product image and shows the generated preview in that same image area. The Shopify app embed—the theme-wide on/off switch—enables this behavior; it is not a block you place in the page layout.

Looksy also offers an optional **Virtual Fitting Room** for multi-product outfit building. That separate storefront page uses a Shopify app block. It does not replace the automatic product-image experience on normal product pages.

## What Looksy does

Use these docs to install Looksy, choose and configure eligible products, turn on the storefront experience, test shopper features, connect integrations, review usage and analytics, and troubleshoot known issues. Begin with a small product set and verify the experience on your own storefront before making it broadly available.

## Use the docs by task

- **Product setup:** Choose products, configure imagery and sizes, create model images, and share direct try-on links.
- **Storefront setup:** Turn on the app embed, adjust the product-image button if needed, and test desktop and mobile storefronts.
- **Shopper features:** Configure size recommendations, Complete the Look, and Live Video Try-On.
- **Virtual Fitting Room:** Add the optional Fitting Room page and choose its catalog or merchant-curated recommendations.
- **Integrations:** Connect Klaviyo or Shopify Flow, and check the current availability of Gorgias.
- **Settings and developer tools:** Set future-product defaults, usage limits, shopper access, API keys, and webhooks.
- **Analytics:** Review the metrics available in Looksy and use the experiment guides without assuming a guaranteed outcome.
- **Troubleshooting:** Check button visibility, product eligibility, image inputs, and device/browser details in a consistent order.
